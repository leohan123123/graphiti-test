## 📝 使用说明

### 启动系统
```bash
# 1. 启动后端服务
cd backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# 2. 启动前端服务
cd bridge-knowledge-platform/frontend
npm run dev -- --port 5173 --host 0.0.0.0

# 3. 确保Ollama和Neo4j服务运行
```

### 系统监控
```bash
# 运行系统监控脚本
./monitor_system.sh

# 运行完整系统测试
python test_complete_system.py
```
